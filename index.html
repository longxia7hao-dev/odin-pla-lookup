本工具為單機速查介面，內建資料來自公開來源並以 ODIN Worldwide Equipment Guide（WEG）常見欄位整理。 正式訓練與引用請以 ODIN 官方內容為準。按 / 快速聚焦搜尋框。

匯入 ODIN／自訂 JSON

可貼上 JSON，或選擇檔案。格式支援：{ "items": [ {...} ] } 或直接陣列。 欄位請對齊 data/schema.example.json。匯入資料會存在瀏覽器本機（localStorage）。

使用說明

  • 單機開啟：直接雙擊 index.html，或執行 ./start.sh 以本機伺服器開啟。
  • 快速搜尋：支援中文名、英文名、型號（如 ZTZ-99A）、別名與標籤；多關鍵字以空白分隔（AND）。
  • 分類：武器／裝備／載具，對應步兵武器、系統裝備、坦克／飛機／艦艇等載具。
  • 軍種篩選：可依陸軍／海軍／空軍／火箭軍／通用過濾。歸類原則：艦艇潛艦與海航／陸戰隊＝海軍；戰機轟炸機與空降兵、戰略防空飛彈（HQ-9／S-300／S-400）＝空軍；陸射彈道／巡航飛彈＝火箭軍；潛射彈道飛彈（JL）＝海軍；步兵武器與單兵裝備跨軍種＝通用；其餘地面裝備＝陸軍。詳細服役單位見各條目「服役單位」欄。
  • 資料出處（引用）：諸元精修條目在詳情頁附可點擊出處連結（如 CSIS Missile Threat、FAS／DoD 中國軍力報告、ONI 艦級命名等）；尚未精修者標示「待補引用」。
  • ODIN 整合：內建資料以 WEG 風格欄位呈現；可將你從 ODIN 整理的 JSON 匯入合併。每筆可加 odin_url 連到官方條目。
  • 繁中翻譯：介面與條目以繁體中文為主，英文名稱與備註並列。
  • 照片：多數條目附 Wikimedia／維基百科公開授權圖片（需連網載入）；缺圖顯示示意占位圖。
  • 資料位置：內建 js/equipment-data.js（可用 tools/build_catalog.py 重建）；匯入內容在瀏覽器 localStorage。
  • 美方權威核對：標「美方核對」者，裝備形式／美方型號／射程帶已對照 DoD《中國軍力報告》2025、FAS CSS 編號、美軍艦級公開命名等；其餘標「待核對」。
  • ODIN：欄位結構對齊 Worldwide Equipment Guide；即時全文需可連 odin.t2com.army.mil。
  • 圖片來源：條目照片多取自 Wikimedia Commons／維基百科（CC BY-SA 或公有領域）；部分冷門型號暫無公開自由圖，以示意占位。
